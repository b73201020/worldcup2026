# 🚀 快速開始指南

## 應用程式概述

這是一個淘汰賽預測應用，採用**樹狀括號式結構**呈現 2026 FIFA 世界盃，5 個階段從 16 強一路預測到冠軍。

## 主要特色

### ✨ 簡單易用
- **只預測勝隊** - 無需輸入比分
- **點擊即選** - 直接點國家旗幟/名稱就選定，不用跳出視窗
- **自動推進** - 勝者自動填入下一輪對戰
- **隨時修改** - 點已選定的隊伍可取消；點另一隊可切換，後續受影響的輪次會自動清除
- **友善列印** - 一鍵輸出 A4 直式樹狀圖

### 📊 賽制結構
```
16強賽        8強賽       準決賽      決賽      冠軍
(8場比賽)    (4場比賽)   (2場比賽)   (1場比賽)  (1支隊伍)

8支隊伍 ──┐
          ├─→ 4支隊伍 ──┐
8支隊伍 ──┘             │
                        ├─→ 2支隊伍 ──┐
8支隊伍 ──┐             │             │
          ├─→ 4支隊伍 ──┘             ├─→ 冠軍
8支隊伍 ──┘                           │
                                      │
                        （對稱另一側）─┘
```

## 如何使用

### 1️⃣ 打開應用程式
```
http://localhost:8000
```

### 2️⃣ 選擇勝者

**16 強賽階段**
- 頁面最左側顯示 8 場比賽
- 每場比賽顯示兩支隊伍（國旗 + 名稱）
- **直接點擊你預測的勝隊**（不需要先點卡片再跳視窗）

**結果**
- ✓ 勝者會高亮顯示（綠色背景）
- 勝者自動填入下一輪（8強賽）對應的位置

### 3️⃣ 依序預測 8強賽 → 準決賽 → 決賽
- 當該場兩支隊伍都確定後，卡片才會變成可點擊
- 點擊你預測的勝隊，自動推進到下一輪
- 一路預測到決賽，冠軍就會顯示在最右側

### 4️⃣ 修改或取消預測
- **想取消**：再點一次目前已選定（綠色）的隊伍
- **想改選**：直接點另一支隊伍
- 兩種操作都會自動清除後續已經推進過的輪次資料，不會留下錯誤的殘留結果

## 介面詳解

### 16強賽 → 8強賽 → 準決賽 → 決賽 → 冠軍
```
┌─ 16強賽 ─┐     ┌─ 8強賽 ─┐     ┌─ 準決賽 ─┐   ┌─ 決賽 ─┐   ┌─ 冠軍 ─┐
│ 🇨🇦加拿大│─┐   │ 待定    │─┐   │  待定    │─┐ │ 待定   │─┐ │        │
│ 🇲🇦摩洛哥│ │   │         │ │   │          │ │ │        │ │ │ 待定   │
└──────────┘ ├──→└─────────┘ ├──→└──────────┘ ├→└────────┘ ├→└────────┘
┌──────────┐ │                │                │
│ 🇵🇾巴拉圭│─┘                │                │
│ 🇫🇷法國  │                  │                │
```

- 卡片之間用連接線畫成真正的樹狀結構，後面輪次的卡片會自動垂直置中對齊前面兩場比賽
- 只有兩支隊伍都確定時，卡片才可點擊

## 按鈕功能

### 🖨️ 列印
- 開啟瀏覽器列印預覽
- 自動切換為 A4 直式版面，維持與螢幕相同的樹狀結構（含連接線），只是縮小尺寸讓 5 欄能印在一頁寬度內
- 按鈕、通知、彈窗等不會被印出來

### 💾 保存
- 預測每次選擇後就會自動保存，此按鈕用於手動再次確認保存
- 顯示「預測已保存」通知

### 🔄 重置
- 清除所有預測，回到初始狀態（只有 16 強隊伍）
- **無法復原** - 會先跳出確認視窗

## 常見操作

### 修改單一預測
- 點擊目前已選定的隊伍 → 取消該選擇
- 或直接點另一支隊伍 → 切換勝者
- 後續已經推進的輪次會自動被清除，避免資料不一致

### 查看進度
- 左上角標題：⚽ 2026 FIFA 世界盃預測
- 所有預測即時反映在各欄
- 綠色 ✓ 表示已完成預測

### 保存與恢復
- 預測自動保存到 LocalStorage
- 關閉瀏覽器後重新打開仍保留
- 清除瀏覽器資料（LocalStorage）會遺失

## 常見問題

### Q: 為什麼某些比賽卡片不能點擊？
A: 當對手隊伍還未確定時（來自前一輪還未預測），該比賽卡片為灰色且不可點擊。

### Q: 我的預測會自動保存嗎？
A: 是的，每次選擇/取消/切換勝者時都會自動保存到 LocalStorage。

### Q: 可以修改已經選好的預測嗎？
A: 可以。點擊已選定的隊伍可取消，或直接點另一支隊伍切換，後續輪次會自動清除更新。

### Q: 可以重新開始嗎？
A: 可以。點擊「🔄 重置」按鈕清除所有預測。

### Q: 預測會在哪裡保存？
A: 保存在瀏覽器的 LocalStorage 中，不會上傳到任何伺服器。

### Q: 可以列印我的預測嗎？
A: 可以，點擊「🖨️ 列印」，會依 A4 直式版面輸出與畫面一致的樹狀圖。
