# ⚽ 2026 FIFA 世界盃預測 - 淘汰賽預測器

一個互動式的淘汰賽預測應用，讓你輕鬆追蹤 2026 FIFA 世界盃的賽程，從 16 強一路預測到冠軍！

## 功能特色

✨ **核心功能**
- 📊 樹狀括號式賽制呈現（16強 → 半準決賽 → 決賽 → 冠軍）
- 🎯 簡單的勝者預測（只需選擇哪隊晉級）
- 🔄 自動更新下一輪對手（勝者自動進入下一輪）
- 💾 自動保存到瀏覽器本地
- 📱 響應式設計（支援所有裝置）

## 使用方式

### 開始預測

1. **打開應用程式**
   ```
   http://localhost:8000
   ```

2. **選擇勝者**
   - 當兩支隊伍都已確定時，點擊該比賽卡片
   - 選擇你預測的勝者
   - 勝者會自動進入下一輪

3. **追蹤進度**
   - 左側：16 強賽的 8 場比賽
   - 中央：半準決賽的 4 場比賽（顯示自動推進的隊伍）
   - 右側：決賽的 2 場比賽
   - 最右：冠軍顯示

### 保存與重置
- **💾 保存** - 預測自動保存到瀏覽器
- **🔄 重置** - 清除所有預測（無法復原）

## 技術特性

### 淘汰賽邏輯
```
16強賽 (8場)
├─ 匹配1&2 → 半決賽1
├─ 匹配3&4 → 半決賽2
├─ 匹配5&6 → 半決賽3
└─ 匹配7&8 → 半決賽4

半準決賽 (4場)
├─ 半決賽1&2 → 決賽1
└─ 半決賽3&4 → 決賽2

決賽 (2場)
├─ 決賽1 贏家 → 冠軍
└─ 決賽2 贏家 → 亞軍
```

### 技術棧
- **HTML5** - 語意標記
- **CSS3** - 響應式設計、深色模式支援
- **Vanilla JavaScript** - 無框架，輕量級
- **LocalStorage** - 用戶端資料持久化

## 常見問題

**Q: 我的預測會遺失嗎？**
A: 預測儲存在你的瀏覽器本地。清除瀏覽器資料時會遺失。

**Q: 可以修改已預測的結果嗎？**
A: 可以。點擊已完成的比賽，重新選擇勝者即可。

**Q: 下一輪對手何時出現？**
A: 當上一輪的對手全部確定後，下一輪對手會自動顯示。

**Q: 可以在多個裝置同步嗎？**
A: 目前不支援。數據只儲存在單一裝置的瀏覽器中。
